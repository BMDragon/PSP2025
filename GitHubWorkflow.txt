Before working: git pull origin main

Checkout a branch: git checkout -b <your name>
* if branch already exists: git checkout [branch name]
* branch name will just be your name

Work locally

To add changes to your branch:
    1. git add .
    2. git commit -m "<message detailing changes>"
    3. git push

To merge your branch into main:
    1. Go to the repository on GitHub and go to the "Pull requests" tab
    2. Click the green button "New pull request"
    3. Resolve merge conflicts and pull into main
